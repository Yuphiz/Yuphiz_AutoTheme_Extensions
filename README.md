# Yuphiz_自动主题_扩展

本页是 Yuphiz_自动主题 的扩展支持

自己根据需要选择下载


使用方法：
放到Yuphiz_自动主题下面的【扩展】文件夹，有些脚本处于禁用状态，重命名去掉前面的双下划线(__)即可以使用

禁用：脚本或者脚本文件夹名称前面加双下划线(__)即可以禁用，想方便可以在Yuphiz_自动主题下面的【扩展】文件夹新建一个【__禁用】文件夹，禁用的文件夹丢进里面就好

设置：有json的改json，没有json的改ps1文件
